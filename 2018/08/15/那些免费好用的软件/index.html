那些免费好用的软件

发表于 2018-08-15 | 分类于 杂项

自从读了吴军老师的《见识》一书，再一次让我觉得时间是何尝宝贵。引用里面的一句话，拒绝做伪工作者。
给大家推荐一些好用的软件，能提升一定的工作效率，节约宝贵时间

1.电子书管理神器：Calibre

Kindle 用户耳熟能详的电子书管理神器 Calibre，也是一款高质量的开源软件。它可以管理几乎所有的电子书格式，与电子书有关的所有操作都可以在 Calibre 一个软件中完成。图书管理、格式转换、新闻获取、电子书编辑……几乎无所不能
GitHub地址
Calibre官网下载

2.优质开源启动器：Wox

Wox 是一款快捷启动器，类似于 macOS 的 Alfred 和 Windows 的 Listary。它可以启动程序、执行命令行、搜索网页，还可以自定义各种功能。
GitHub地址
Wox下载地址

3.全局鼠标手势：WGestures

WGestures 可以让鼠标操作更简单，把琐碎的操作归结为大量的鼠标手势操作。比如选择文字后，上下一滑就可以进入百度搜索。还增加了触发角、摩擦边等有意思的功能。
相比 Wox、Listary 提倡的纯键盘操作，纯鼠标操作或许也是个不错的选择。
GitHub 地址
WGestures官网地址

4.本地密码管理软件：KeePassX

KeePassX 是一款本地密码管理软件，它是 KeePass 的一个移植版本，如今已经是全平台密码管理软件了。
整体很简单，密码方面的功能够用，采用了 AES256 和 Twofish 算法来加密密码。支持自动排序、快速备份等功能。
GitHub地址
KeePassX官网下载

5.实现一键快速预览：QuickLook

QuickLook 是一款免费的轻量级快速预览工具。它能够在不运行关联程序的情况下，通过敲击空格键快速预览文件内容。
你可以在Microsoft Store免费下载

6.截图神器：Snipaste

除了自动检测边界及文本、形状、箭头、涂鸦、高亮、马赛克等强大标注功能外，Snipaste 还能让你将截图或剪贴板内容直接固定在屏幕上，方便随时查看。
你可以在Microsoft Store免费下载

7.搜得比什么都快：Listary

你可以在任意文件资源管理器窗口通过键盘输入直接调用 Listary 在本目录下进行搜索，并通过右键单击执行各种命令。还可以出现在文件选择对话框下，方便快速搜索指定路径，并快速切换目录。此外，你还可以用 Listary 搜索网页、打开应用等。
Listary官网地址

8.全平台云剪贴板：快贴

跨平台进行黏贴复制，有云保存功能，而且可以将需要经常复制黏贴的文字图片收藏。
快贴官网地址
